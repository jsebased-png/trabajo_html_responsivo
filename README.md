# 🌐 Proyecto MyPortfolio - Sesión 6 CSS

Este proyecto es una práctica de maquetación web utilizando **HTML5** y **CSS3**.  
El objetivo es construir una página moderna y responsiva que incluya navegación, secciones informativas y una galería de imágenes tipo catálogo.

---

## 📂 Estructura del Proyecto

- **index.html** → Página principal con:
  - Barra de navegación fija.
  - Sección *Hero* con título y descripción.
  - Sección de habilidades con tarjetas.
  - Perfil del estudiante con imagen y descripción.
  - Etiquetas de tecnologías usadas.
  - Footer con créditos.

- **galery.html** → Página secundaria con:
  - Barra de navegación.
  - Contenedor de tarjetas (`.CARD`) que muestran imágenes responsivas simulando un catálogo.

- **css/styles.css** → Archivo de estilos con:
  - Tipografía y colores base.
  - Estilos para la barra de navegación.
  - Diseño del *Hero* y botones.
  - Tarjetas de habilidades y perfil.
  - Galería con tarjetas responsivas.
  - Media queries para adaptar el diseño en diferentes pantallas.

---

## 🎨 Características Principales

- **Diseño responsivo**:  
  Uso de `flexbox` y `grid` con *media queries* para adaptar la galería y las secciones a distintos tamaños de pantalla.

- **Tarjetas con efectos**:  
  - Fondo con degradados (`linear-gradient`).  
  - Sombras (`box-shadow`).  
  - Animación al pasar el cursor
