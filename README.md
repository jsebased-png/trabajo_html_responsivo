# 🌐 Proyecto MyPortfolio - Sesión 6 CSS

Este proyecto es una práctica de maquetación web utilizando **HTML5** y **CSS3**.  
El objetivo es construir una página moderna y responsiva que incluya navegación, secciones informativas y una galería de imágenes tipo catálogo.

---

## 📂 Estructura del Proyecto

- **index.html** → Página principal con:
  - Barra de navegación fija.
  - Sección *Hero* con título y descripción.
  - Sección de habilidades con tarjetas.
  - Perfil del estudiante con imagen y descripción.
  - Etiquetas de tecnologías usadas.
  - Footer con créditos.

- **galery.html** → Página secundaria con:
  - Barra de navegación.
  - Contenedor de tarjetas (`.CARD`) que muestran imágenes responsivas simulando un catálogo.

- **css/styles.css** → Archivo de estilos con:
  - Tipografía y colores base.
  - Estilos para la barra de navegación.
  - Diseño del *Hero* y botones.
  - Tarjetas de habilidades y perfil.
  - Galería con tarjetas responsivas.
  - Media queries para adaptar el diseño en diferentes pantallas.

---

## 🎨 Características Principales

- **Diseño responsivo**:  
  Uso de `flexbox` y `grid` con *media queries* para adaptar la galería y las secciones a distintos tamaños de pantalla.

- **Tarjetas con efectos**:  
  - Fondo con degradados (`linear-gradient`).  
  - Sombras (`box-shadow`).  
  - Animación al pasar el cursor (`hover` con `transform: scale`).  
  - Imágenes con `object-fit: cover` para llenar el cuadro sin deformarse.

- **Estética moderna**:  
  - Colores vivos y contrastantes.  
  - Bordes redondeados.  
  - Botones con transiciones suaves.  

---

## 🚀 Cómo usar

1. Clona o descarga el proyecto.
2. Abre `index.html` en tu navegador para ver la página principal.
3. Accede a la galería desde el menú de navegación o directamente abriendo `galery.html`.
4. Personaliza las imágenes cambiando las URLs en el HTML (`https://picsum.photos/...`).

---

## 📱 Responsividad

- **Pantallas pequeñas (<768px)** → Una columna en la galería.  
- **Pantallas medianas (≥768px)** → Dos columnas.  
- **Pantallas grandes (≥1024px)** → Tres columnas.  

---

## 🛠️ Tecnologías utilizadas

- **HTML5** para la estructura semántica.  
- **CSS3** para estilos, transiciones y diseño responsivo.  
- **Google Fonts (Poppins)** para tipografía moderna.

# 🌐 MyPortfolio Project - CSS Session 6

This project is a web layout practice using **HTML5** and **CSS**.  
The goal is to build a modern, responsive webpage that includes navigation, informative sections, and an image gallery styled as a commercial catalog.

---

## 📂 Project Structure

- **index.html** → Main page with:
  - Sticky navigation bar.
  - *Hero* section with title and description.
  - Skills section with cards.
  - Student profile with avatar and description.
  - Tags showcasing technologies used.
  - Footer with credits.

- **galery.html** → Secondary page with:
  - Navigation bar.
  - Image cards (`.CARD`) simulating a responsive product catalog.

- **css/styles.css** → Stylesheet with:
  - Base typography and colors.
  - Navigation bar styling.
  - Hero section and button design.
  - Skill cards and profile layout.
  - Responsive gallery with image cards.
  - Media queries for different screen sizes.

---

## 🎨 Key Features

- **Responsive design**:  
  Uses `flexbox` and `grid` with *media queries* to adapt layout across devices.

- **Interactive cards**:  
  - Gradient backgrounds (`linear-gradient`).  
  - Shadows (`box-shadow`).  
  - Hover animations (`transform: scale`).  
  - Images styled with `object-fit: cover` to fill the card without distortion.

- **Modern aesthetics**:  
  - Vibrant, contrasting colors.  
  - Rounded corners.  
  - Smooth button transitions.  

---

## 🚀 How to Use

1. Clone or download the project.
2. Open `index.html` in your browser to view the main page.
3. Navigate to the gallery via the menu or open `galery.html` directly.
4. Replace the placeholder image URLs (`https://picsum.photos/...`) with your own product images.

---

## 📱 Responsiveness

- **Small screens (<768px)** → One column in the gallery.  
- **Medium screens (≥768px)** → Two columns.  
- **Large screens (≥1024px)** → Three columns.  

---

## 🛠️ Technologies Used

- **HTML5** for semantic structure.  
- **CSS3** for styling, transitions, and responsive layout.  
- **Google Fonts (Poppins)** for modern typography.
